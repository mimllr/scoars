# Scoar

[![Netlify Status](https://api.netlify.com/api/v1/badges/7c245783-1fe1-4024-9681-1284b89f8bee/deploy-status)](https://app.netlify.com/projects/scoars/deploys)

A live NHL scoreboard built as a single-page app. Pulls real-time scores, period breakdowns, and goal timelines from ESPN's API, refreshing every 30 seconds. Designed to look great on mobile.
